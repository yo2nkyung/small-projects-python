# Simple Quiz program

A simple python quiz program that asks questions and evaluates user answers

## Overview

사용자에게 질문하고 답변을 입력받은 후, 정답 여부를 평가하고 점수를 계산하여 결과를 알려주는 간단한 퀴즈 프로그램

### 주요 기능

- 질문에 대한 사용자 입력 받기
- 사용자의 답변 정답과 비교하여 평가
- 퀴즈 종료 후 총 점수 출력
- 질문과 정답은 json 파일에서 읽어오기

### 필수 설치 조건
- python 3.x
