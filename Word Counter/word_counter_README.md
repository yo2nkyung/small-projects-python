# Word Counter
A simple Python project to count word frequencies in a text file.

## 프로젝트 개요
주어진 텍스트 파일에서 단어의 빈도수를 계산하고, 결과를 출력 및 파일로 저장하는 프로그램

### 주요 기능
- 텍스트 파일에서 단어를 추출하여 빈도수를 계산
- 특수 문자와 구두점 처리를 포함한 단어 추출
- 결과를 정렬하여 출력
- 단어 빈도수를 CSV 파일로 저장


## 설치 방법

### 필수 설치 조건
- Python 3.x
- 정규 표현식을 사용하기 위한 `re` 모듈 (Python 내장 모듈)
